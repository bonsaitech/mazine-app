# Mazine Analytics

Dashboard IA d'analyse de performance TikTok — propulsé par GPT-4o.

## Stack
- Node.js + Express (serveur statique)
- HTML / CSS / JS vanilla (frontend)
- OpenAI GPT-4o (analyse IA)
- Chart.js, SheetJS, PapaParse

## Lancer en local
```bash
npm install
npm start
```
Ouvre http://localhost:3000

## Déploiement Railway
Le projet se déploie automatiquement via GitHub → Railway.
Aucune variable d'environnement requise (la clé OpenAI est saisie côté client).
